# User Documentation

This directory contains documentation for users, operators, and administrators of the Gen AI system.

## Available Guides

- [Logging Configuration](./admin/logging/README.md) - How to configure and enable logging across gen-ai components
- [RBAC Configuration](./admin/rbac/README.md) - How to configure RBAC endpoint filtering for LlamaStack

## Documentation Structure

- `admin/` - Administrative and operational documentation
  - `logging/` - Logging configuration and troubleshooting guides
  - `rbac/` - RBAC endpoint filtering configuration for LlamaStack

## Contributing

When adding new user-facing documentation:
1. Create a new markdown file in the appropriate subdirectory
2. Use descriptive, kebab-case filenames (e.g., `deployment.md`, `troubleshooting.md`)
3. Update this README to include a link to your new documentation
4. Follow the existing documentation style and format
